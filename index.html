Framework

Un framework est un espace de travail modulaire.

C'est un ensemble de bibliothèques et de conventions permettant de :

  1. Simplifier le travail des développeurs informatiques
  2. Offrir une architecture prête à l’emploi.
  3. Développer des applications rapidement…

Parmis les frameworks web on trouve :

  • React JS
  • Angular
  • Node JS
  • Django…
